# 🌌 KINETIC: The Gamified Social Frontier

Welcome to **Kinetic**, a high-octane social media platform designed for the next generation of digital interaction. Kinetic merges traditional social features with a deep gamification engine, turning every post, message, and interaction into a quest for XP and global dominance.

![Kinetic Preview](frontend/src/assets/hero.png)

## 🚀 Key Features

- **⚡ Gamified Core**: Earn XP for everything you do. Post content, comment on theories, and maintain streaks to level up your profile.
- **🔥 Streaks & Milestones**: Build daily streaks with your squad. Hit milestones to unlock special notifications and rewards.
- **💹 Global Leaderboard**: Compete with others in real-time. Can you reach the #1 rank in the sector?
- **💬 Interstellar Messaging**: Real-time chat with custom avatars and seamless interaction.
- **🔔 Kinetic Alerts**: Stay informed with a dynamic notification system that tracks every "Sector Update" from likes to rank changes.
- **📸 Breaking News**: A dedicated feed for the most impactful "High-XP" content across the platform.

## 🛠️ Technology Stack

### Frontend
- **Framework**: React 18 with Vite
- **Styling**: Tailwind CSS (Deep Space Theme)
- **Icons**: Lucide React
- **Routing**: React Router DOM

### Backend
- **Engine**: Express.js (Node.js)
- **Language**: TypeScript
- **Database**: SQLite (LibSQL)
- **ORM**: Drizzle ORM
- **Auth**: JWT (JSON Web Tokens) & Bcrypt

## 🛸 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. **Clone the Sector**
   ```bash
   git clone https://github.com/Nikhil7565/social-media-platform.git
   cd social-media-platform
   ```

2. **Initialize Backend**
   ```bash
   cd backend
   npm install
   # Create a .env file based on .env.example (or use defaults)
   npm run dev
   ```

3. **Initialize Frontend**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```

4. **Access the System**
   Navigate to `http://localhost:5173` to enter the platform.
